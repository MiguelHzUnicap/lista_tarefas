# Lista de Tarefas - Next.js 15

Projeto desenvolvido para praticar testes unitários utilizando Next.js 15, TypeScript, Jest e Testing Library.

## Funcionalidades

- Listagem de tarefas simulando uma API
- Adição de novas tarefas
- Hook personalizado para contagem de tarefas
- Testes unitários para componentes e hooks

## Tecnologias

- Next.js 15
- React
- TypeScript
- Jest
- Testing Library

## Instalação

Clone o repositório

git clone

Instale as dependências

npm install

Execute o projeto

npm run dev

## Executar testes

npm test
